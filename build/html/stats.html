We re-use the One-sample t-test test statistic from above

calc_tstat <- function(x, mu=0){
    (mean(x)-mu) / (sd(x)/sqrt(length(x)))

Maximum t-test

maxt.test <- function(x, g, n=10^5)
{
    # Calculate test statistics of the group that is most different from the others
    m <- mean(x)
    stat <- tapply(x, g, calc_tstat, mu=m) %>%
            abs() %>%
            max()
    gsize = table(g)
    
    
    # Fill z with n random values that follow a T-distribution. This has to
    # be done sepparetly for each group due to different group sizes
    
    z <- NULL
    for(k in 1:length(gsize)){
        df = gsize[k]-1
        random_t = rt(n=n, df)
        z <- cbind(z, random_t )
    }

    # z now is a n x length(gsize) matrix, again we take the maximum absolute value
    z <- abs(z) %>%
      rowMaxs()
    # z <- z[cbind(1:n,max.col(z))] alternative implementation for rowMaxs()

    # Compare test statistics to the randomly generated T-distributed z values. 
    # The mean is our p-value (works because TRUE = 1 and FALSE = 0)
    pval=mean(z>stat)
    
    # Output our test results
    retval <- list(tstat=stat, 
                   pval=pval,
                   name="Maximum t-test")
    class(retval) <- "ttest"
    retval
}

Permutation maximum T-test

The sampled values are randomly reasigned between groups. Under the null hypothesis, the underlying distributions for each group are the same, therefore it should make no difference which group an observation gets assigned. In the optimal case all possible group/sample combinations would be tested.

  • Advantage: we do not need to know the distribution of the statistic under the null
  • Disadvantage: much slower
maxt.ptest <- function(x, g, n=10^4)
{
    m <- mean(x)
    stat <- tapply(x, g, calc_tstat, mu=m) %>%
            abs() %>%
            max()
    
    # Instead of generating n T-distributed values, we generate n tests-statistics
    # from our sample data randomly reassigned between our sample groups.
    z <- double(n)
    for(k in 1:n){
        g1 <- sample(g)
        z[k] <- tapply(x, g1, calc_tstat, mu=m) %>%
                abs() %>%
                max()
    }
    
    # The computation of the p-value is the same as for the maximum t-test    
    retval <- list(tstat=stat, 
                   pval=mean(z>stat),
                   name="Permutation maximum t-test")
    class(retval) <- "ttest"
    retval
}

Permutation test

permtest <-
function(x, g, n=1000, conf.level=0.95){
  
  lg <- levels(g)
  mdiff <- mean(x[g==lg[1]]) - mean(x[g==lg[2]])
  z <- double(n)
  for(k in 1:n){
    g1 <- sample(g)
    z[k] <- mean(x[g1==lg[1]]) - mean(x[g1==lg[2]])
  }
  
  alpha <- 1-conf.level
  conf.int <- quantile(z, prob=c(alpha/2, (1-alpha)/2))
  attr(conf.int, "conf.level") <- conf.level
  
    
  estimate <- c(mean(x[g==lg[1]]),
                mean(x[g==lg[2]]))
  names(estimate) <- c(paste("mean of", lg[1]),  
                       paste("mean of", lg[2]))
  
  null.value <- 0
  names(null.value) <- paste(paste("mu.", lg[1], sep =""),
                             "-", 
                             paste("mu.", lg[2],  sep = ""))
  
  data <- paste(deparse(substitute(x)),             
                "by",
                deparse(substitute(g)))
          
  retval <- list(null.value  = null.value,
                 alternative = paste("True mu.", lg[1], " - mu.", lg[2], " is not equal to 0.", sep =""), 
                 method      = "Permutation test for difference in means",
                 estimate    = estimate,
                 data.name   = data,
                 statistic   = c("Difference of means (mdiff)"=mdiff),
                 parameters  = c(nrep=n),
                 p.value     = mean(abs(z)>abs(mdiff)),                 
                 conf.int    = conf.int)
  
  class(retval) <- "htest"
  return(retval)
}

h-test class

P-value adjustment (Multiple testing correction)

re-calculating probabilities obtained from a statistical test which was repeated multiple times. In order to retain a prescribed family-wise error rate α in an analysis involving more than one comparison, the error rate for each comparison must be more stringent than α.

  • Bonferroni correction
    • multiply all p-values by number of tests
    • simplest and most conservative method to control the familywise error 3 rate.
  • Holmes correction, allways better than Bonferroni
    • Under the null the p-values should lie on a line with slope 1/N ->
    • multiply the smallest p-value with N, the second with N-1 etc.
  • Both are valid under arbitrary assumptions
  • There are more powerfull methodes than Holms, but all of them make assumptions on the single tests like f.e. independence or non-negative association.

  • Bonferonni corrected p values 0.29364, 0.44991, 1, 1, 1, 1, 1, 1…
  • Holmes corrected p values 0.29364, 0.44541, 1, 1, 1, 1, 1, 1…

Basics & Theory

Hypothesis

H0: Null Hypothesis states that the population parameter (population mean μ or proportion p) is equal to some value

  • reject H0: “I have enough evidence to proof this statement wrong”
  • fail to reject H0: “I do not have enough evidence to proof this statement wrong therefore i accept the alternative”

H1 : States that the parameter (μ, p) has a value inequal (smaller, greater, not equal) to H0.

  • H0 is rejected: H1 is true
  • H0 is not rejected: I can’t proof H1 true (experiment is inconclusive)

If you want to proof a statement you must state it as H1

Examples:

H0 : p = .80 (80% of all swans are white)
H0 : μ = 10 (The average speed of a pidgeon is 10 m/s)
H1 : p < .80 (less than 80% of all swans are white)
H1 : μ < 10 (The average speed of a pidgeon is less than 10 m/s)

Recipe

  1. original claim about proportion or mean?
  2. state claim symbolically
  3. state opposite of claim symbolically
  4. state H0 (equality) and H1 (inequality).

Example:

“the mean of fluid is at least 12oz in a can”


$$ \begin{align*} claim: \mu & \geq 12 &\rightarrow H_0: \mu = 12 \\ opp: \mu &< 12 &\rightarrow H_1: \mu < 12 \end{align*} $$

We can only proof that there is less than 12 oz in a can, the claim is not stated correctly

Errors

  • Type 1 Error α… Level of significance. Probability that Hypothesis is rejected despite beeing true.
  • Type 2 Error β… False negative. Probability that a false H0 is not rejected
  • Statistical Power 1 − β… The probability that a wrong H0 is correctly rejected. Also called sensitivty.

Z- and T-Tests

Population standard deviation known Population standard deviation not known

$$Z = \frac{\bar x - \mu}{\sigma / \sqrt{n}}$$
$$T = \frac{\bar x - \mu}{s / \sqrt{n}}$$

T-test assumptions

  • random sample
  • σ known
  • n > 30 or population normally distributed

R-test assumptions

  • random sample
  • σ not know
  • n < 30 or population normally distributed

Differences:

  • Z-tests follow a normal distribution, while T-test follows a Student’s T-distribution 4.
  • For large n T and Z tests become very similar
  • p-values for T-tests harder to find out if you do it manually

T-test for location overview

Uses:

  • A one-sample location test of whether the mean of a population has a value specified in a null hypothesis.
  • Unpaired two-sample location test (Student’s t-tests)
    • Null hypothesis: Means of two populations are equal.
  • Paired two-sample location test:
    • Null hypothesis: Difference between two responses measured on the same statistical unit has a mean value of zero.
    • For example, suppose we measure the size of a cancer patient’s tumor before and after a treatment. If the treatment is effective, we expect the tumor size for many of the patients to be smaller following the treatment. This is often referred to as the “paired” or “repeated measures” t-test
  • A test of whether the slope of a regression line differs significantly from 0.

The test-statists

Generic test-statistic: $T = \frac{Z} {(s/\sqrt{n})}$

  • Z A function sensitive to the alternative hypothesis (i.e., its magnitude is larger when the alternative hypothesis is true)
  • $\bar{X}$ :sample mean from a sample X1, X2, ..., Xn,
  • n: sample size,
  • s: sample standard deviation (= $\sqrt{\text{sample variance}}$),
  • σ: population standard deviation.

Once a t value is determined, a p-value can be found using a table of values from Student’s t-distribution.

Assumptions:

  • X follows a normal distribution with mean mu and variance σ2
  • S2 follos a chi-square distribution with p degrees of freedom under the null hypothesis
    • chi-square distribution the distribution of a sum of the squares of k independent standard normal random variables.
    • degrees of freedom: the number of values in the final calculation of a statistic that are free to vary
      • minimum number of independent coordinates that can specify the position of the system completely
      • equal to the number of independent scores that go into the estimate minus the number of parameters used as intermediate steps in the estimation of the parameter itself (i.e., the sample variance has N-1 degrees of freedom, since it is computed from N random scores minus the only 1 parameter estimated as intermediate step, which is the sample mean).
  • Z and s are independent

One-sample t-test

Testing the null hypothesis that the population mean is equal to a specified value μ0. The degree of approximation will depend on how close the parent population is to a normal distribution and the sample size, n


$$ T = \frac{\overline{x} - \mu_0}{s/\sqrt{n}} $$

  • $\overline{x}$ is the sample mean, assumed to be normal distributed (see central limit theorem5)
  • s is the sample standard deviation
  • n is the sample size.
  • The degrees of freedom used in this test are n − 1.
  • Parent population does not need to be normally distributed

Calculate the p-value from the test statistc

p-value is the probability of obtaining the observed sample results, or a “more extreme” result, when assuming the null hypothesis is actually true.

If the p-value is equal to or smaller than the significance level (α), it suggests that the observed data are inconsistent with the assumption that the null hypothesis is true, and thus that hypothesis must be rejected and the alternative hypothesis is accepted as true. When the p-value is calculated correctly, such a test is guaranteed to control the Type I error rate to be no greater than α (Type I error: Null hypothesis (H0) is true, but is rejected).

Common misunderstandings about p-values.

  • The p-value is not the probability that the null hypothesis is true, nor is it the probability that the alternative hypothesis is false – it is not connected to either of these. In fact, frequentist statistics does not, and cannot, attach probabilities to hypotheses. Comparison of Bayesian and classical approaches shows that a p-value can be very close to zero while the posterior probability of the null is very close to unity (if there is no alternative hypothesis with a large enough a priori probability and which would explain the results more easily). This is Lindley’s paradox. But there are also a priori probability distributions where the posterior probability and the p-value have similar or equal values.
  • The p-value is not the probability that a finding is “merely a fluke.” Calculating the p-value is based on the assumption that every finding is a fluke, that is, the product of chance alone. Thus, the probability that the result is due to chance is in fact unity. The phrase “the results are due to chance” is used to mean that the null hypothesis is probably correct. However, that is merely a restatement of the inverse probability fallacy, since the p-value cannot be used to figure out the probability of a hypothesis being true.
  • The p-value is not the probability of falsely rejecting the null hypothesis. This error is a version of the so-called prosecutor’s fallacy.
  • The p-value is not the probability that replicating the experiment would yield the same conclusion. Quantifying the replicability of an experiment was attempted through the concept of p-rep.
  • The significance level, such as 0.05, is not determined by the p-value. Rather, the significance level is decided by the person conducting the experiment.
  • The p-value does not indicate the size or importance of the observed effect. The two do vary together, however, the larger the effect, the smaller sample size will be required to get a significant p-value (see effect size).

Equal sample sizes, equal variance

  • The two sample sizes (that is, the number, ‘’n’’, of participants of each group) are equal;
  • It can be assumed that the two distributions have the same variance.

The ‘’t’’ statistic to test whether the means are different can be calculated as follows:


$$ t = \frac{\bar {X}_1 - \bar{X}_2}{s_{X_1X_2} \cdot \sqrt{\frac{2}{n}}} $$

where
$$ \ s_{X_1X_2} = \sqrt{\frac{1}{2}(s_{X_1}^2+s_{X_2}^2)} $$

Equal or unequal sample sizes, equal variance

Used only when it can be assumed that the two distributions have the same variance. (When this assumption is violated, see below.) The ‘’t’’ statistic to test whether the means are different can be calculated as follows:
$$ t = \frac{\bar {X}_1 - \bar{X}_2}{s_{X_1X_2} \cdot \sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} $$

where


$$ s_{X_1X_2} = \sqrt{\frac{(n_1-1)s_{X_1}^2+(n_2-1)s_{X_2}^2}{n_1+n_2-2}}. $$

Equal or unequal sample sizes, unequal variances (Welch test)

Used only when the two population variances are not assumed to be equal (the two sample sizes may or may not be equal) and hence must be estimated separately. The ‘’t’’ statistic to test whether the population means are different is calculated as:


$$t = {\overline{X}_1 - \overline{X}_2 \over s_{\overline{X}_1 - \overline{X}_2}}$$

where


$$s_{\overline{X}_1 - \overline{X}_2} = \sqrt{{s_1^2 \over n_1} + {s_2^2 \over n_2}}. $$

Dependent t-test for paired samples

Used when the samples are dependent; that is, when there is only one sample that has been tested twice (repeated measures) or when there are two samples that have been matched or “paired”. This is an example of a [[paired difference test]].


$$ t = \frac{\overline{X}_D - \mu_0}{s_D/\sqrt{n}}. $$

For this equation, the differences between all pairs must be calculated. The pairs are either one person’s pre-test and post-test scores or between pairs of persons matched into meaningful groups (for instance drawn from the same family or age group: see table). The average (XD) and standard deviation (sD) of those differences are used in the equation. The constant μ0 is non-zero if you want to test whether the average of the difference is significantly different from μ0. The degree of freedom used is n − 1.

Terms

Variance

$\operatorname{Var}(X) = \operatorname{E}\left[(X - \mu)^2 \right] = \sum_{i=1}^n (x_i-\mu)^2 p(x_i)$.

  • p(xi): probability of xi
  • E[…]: expected value (mean)
  • Intepretation:
  • 0: All values are identical
  • small: All data points close to mean
  • high: data points spread out around the mean and from each other

A variance of zero indicates that all the values are identical. a small variance indicates that the data points tend to be very close to the mean (expected value) and hence to each other, while a high variance indicates that the data points are very spread out around the mean and from each other.

Normal Distribution:

  • X: Random Variable (Numerical outcome of a random experiment)
  • μ: population mean,
  • $\bar{x}$: sample mean,
  • σ: poluation standard deviation,
  • s: sample mean.

It is an observed interval (i.e. it is calculated from the observations), in principle different from sample to sample, that frequently includes the parameter of interest if the experiment is repeated. How frequently the observed interval contains the parameter is determined by the confidence level or confidence coefficient.↩

  2. UCL/LCL = CL ± k * S

    • CL = centre line (sample mean)
    • S = standard deviation (appropriate valuation of standard deviation)
    • k = 3 (usually, but can differ)
    ↩

  3. Family wise error rate: Probability of making one or more false discoveries, or type I errors, among all the hypotheses when performing multiple hypotheses tests.↩
